List of working versions

The hydrodynamic model started as a Fortran code in serie for estuaries, which was later modified to be suitable for lake studies. Over the time, researchers have contributed in the evolution of SI3D by adding a water quality model and parallelizing the source code for computational purposes. Due to the variable versions of SI3D-L, we provide the GitHub link to two of the main master source codes available. Following the corresponding section for each version, the GitHub repositories are linked to the corresponding title and symbol.
